# 介護食はじめてノート（介護食アフィリエイトブログ）

「親の介護が始まって、食事に戸惑っている人」向けの入門ブログです。
やわらか食宅配サービスや市販の介護食を紹介して、アフィリエイトで収益化を目指します。
GitHub Pagesで公開する静的サイトなので、サーバー代は0円です。

## なぜこのジャンルか

- 高齢化で読者（必要とする人）は増え続けている
- 子供服・コスメ・ガジェットに比べて、個人で書いている人がかなり少ない
- 宅配食系の広告案件は1件1,000〜3,000円程度と単価が高め（子供服は購入額の数%）
- 「介護食 とは」「介護食 宅配 おすすめ」など、初心者向けキーワードに空きがある

## サイトの場所

- 公開用ファイルはぜんぶ `docs/` フォルダの中にあります
- GitHub Pages の設定: branch `main` / folder `/docs`
- 新リポジトリ名の想定: `kaigoshoku-note`

## ファイル構成

- `docs/index.html` … トップページ（記事一覧）
- `docs/articles/takuhai-erabikata.html` … 記事1（宅配サービスの選び方・収益記事）
- `docs/articles/kubun-toha.html` … 記事2（UDF区分の見方・集客記事）
- `docs/articles/template.html` … 新しい記事を書くときのテンプレート
- `docs/privacy.html` … プライバシーポリシー・免責事項（広告審査に必要）
- `docs/css/style.css` … サイト全体のデザイン

## 収益化までにやること（順番どおりに）

1. **新リポジトリを作って公開する**
   - GitHub で `kaigoshoku-note` リポジトリを作成（Public）
   - この `kaigoshoku-note/` フォルダの中身一式をそのリポジトリに入れる
   - Settings → Pages → Branch: `main` / Folder: `/docs` で公開

2. **アフィリエイトに登録する（無料）**
   - A8.net: https://www.a8.net/ （メインASP。管理画面で「やわらか食」「制限食 宅配」「宅配弁当」で案件を検索）
   - 楽天アフィリエイト: https://affiliate.rakuten.co.jp/ （レトルト介護食の商品リンク用。審査がゆるく最初に使いやすい）
   - Amazonアソシエイト: https://affiliate.amazon.co.jp/ （サイト公開後に申請）

3. **記事のリンクを差し替える**
   - 各記事の `<!-- ▼▼▼ ここにアフィリエイトリンクを貼る -->` の部分の `href="#"` を、広告リンクのURLに置き換える

4. **記事を増やす**
   - `docs/articles/template.html` をコピーして新しいファイル名にする
   - `docs/index.html` の記事一覧にカードを1つ追加する

## 記事を書くときの注意（重要）

このジャンルは健康に関わる「YMYL」と呼ばれる分野で、Googleも読者も信頼性に敏感です。

- 「これで誤嚥が防げる」「病気が良くなる」など、**医療的な断定は絶対に書かない**
- 迷ったら「医師・ケアマネジャー・管理栄養士に相談してください」と添える（記事内の注意ボックスを使う）
- 実際に試したもの・調べたことだけを書く。使っていないものを「使ってよかった」と書かない
- 各ページ冒頭の「アフィリエイト広告を利用しています」表示は消さない（ステマ規制対応）
